Packages¶

utils¶

Il pacchetto utils (o utilities per esteso) raggruppa tutte le funzioni richiamate successivamente da altri script del progetto.

Ogni funzione viene documentata tramite una breve descrizione riguardante l’operazione che svolge, il dato in input e l’output.

preprocessing¶

Il modulo preprocessing è dedicato alle funzioni per l’esplorazione ed elaborazione preliminare del dato, precedente la fase di data modeling.

arima¶

Il modulo arima è dedicato alle funzioni per la parte lineare del progetto, vale a dire per l’implementazione dei modelli ARIMA.

ucm¶

Il modulo arima è dedicato alle funzioni per la parte lineare del progetto, vale a dire per l’implementazione dei modelli a componenti non osservabili (Unobserved Components Models).

machine learning¶

Il modulo machine learnng è dedicato alle funzioni per la parte non lineare del progetto, vale a dire per l’implementazione dei modelli di apprendimento automatico.
